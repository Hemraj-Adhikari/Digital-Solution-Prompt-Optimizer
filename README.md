An advanced, minimalist, and highly efficient web tool designed to generate ultra-realistic, cinematic image prompts for AI generators like Midjourney, DALL-E, and Stable Diffusion. 

##  Features
* **Minimalist Studio UI:** Clean, dark-mode interface built with Tailwind CSS.
* **Professional Styles:** Pre-configured settings for Editorial, Cinematic, and Minimalist Studio photography.
* **Instant Copy:** One-click copy to clipboard functionality.
* **High-End Defaults:** Automatically applies 8k, ultra-realistic, and dramatic lighting modifiers.

##  How to Use
1. Open the `index.html` file in any modern web browser.
2. Enter your subject details.
3. Select your desired Art Style and Lighting.
4. Click **Generate Prompt** and copy the result!
